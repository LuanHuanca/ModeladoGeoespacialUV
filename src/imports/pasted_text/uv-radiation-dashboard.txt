Diseña un maquetado web profesional, editable y coherente en Figma para un sistema académico llamado:

“SISTEMA INTELIGENTE DE MODELADO GEOESPACIAL Y ANÁLISIS PREDICTIVO DE LA EXPOSICIÓN A RADIACIÓN ULTRAVIOLETA”

Contexto del proyecto:
Es una plataforma web orientada al municipio de La Paz, Bolivia, enfocada en analizar, visualizar y predecir niveles de radiación ultravioleta (UV) utilizando datos meteorológicos, satelitales y geoespaciales. El sistema integra información de fuentes como NASA POWER, SENAMHI y datos topográficos/DEM. Además, utiliza procesamiento geoespacial, un modelo predictivo basado en machine learning y un módulo de inteligencia artificial para generar recomendaciones preventivas.

Objetivo del maquetado:
Crear una interfaz moderna, técnica, clara y académica que permita:
1. Consultar niveles de radiación UV por zona.
2. Visualizar mapas interactivos y mapas de calor.
3. Analizar información histórica por fechas y ubicaciones.
4. Generar histogramas y mapas de tendencia.
5. Mostrar recomendaciones preventivas generadas por IA.
6. Gestionar validación y sincronización de datos desde un panel administrativo.
7. Presentar el sistema de manera adecuada para un informe de taller de grado y para un posible prototipo funcional.

Importante:
- No copiar literalmente ni replicar branding oficial de otras plataformas.
- Inspirarse funcionalmente en portales de monitoreo UV, visualizadores cartográficos, dashboards científicos y sistemas de análisis geoespacial.
- El resultado debe ser completamente editable en Figma.
- Generar diseño en formato desktop first.
- Mantener consistencia visual entre todas las pantallas.
- Usar componentes reutilizables, auto layout y estructura ordenada.
- Priorizar legibilidad, jerarquía visual, limpieza y claridad de navegación.

Inspiración funcional:
- Sistemas de monitoreo y pronóstico UV.
- Paneles con mapa, filtros, tarjetas de riesgo y gráficas.
- Interfaces científicas y geoespaciales.
- Dashboards técnicos para análisis ambiental.
- Flujo visual tipo data platform + GIS dashboard + AI assistant.

Estilo visual:
- Profesional, moderno, limpio, tecnológico y académico.
- Sensación de plataforma científica e institucional.
- Diseño elegante, no infantil.
- Evitar saturación visual.
- Usar mucho espacio en blanco, bloques bien definidos y tarjetas informativas.

Paleta sugerida:
- Azul oscuro / azul petróleo como color principal.
- Celeste / azul claro para elementos informativos.
- Blanco o gris muy claro para fondos secundarios.
- Amarillo, naranja, rojo y morado para representar los niveles de radiación UV.
- Verde para estados correctos o datos validados.
- Gris para elementos secundarios y estados inactivos.

Escala UV visual:
Usar una visualización clara de categorías de radiación UV:
- Bajo
- Moderado
- Alto
- Muy alto
- Extremo

Representarlas con badges, barras, tarjetas o leyenda cromática.
Usar mensajes preventivos visibles y comprensibles para el usuario.

1. Usuario ciudadano
2. Investigador

Diseñar la navegación considerando estos tres perfiles.
